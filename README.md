# 🧾 Mini POS App

Simple Point of Sale (POS) application built with C# and Windows Forms.

## 🚀 Features
- 🛒 Add products to cart
- 💰 Automatic total calculation
- 🗑️ Remove items from cart
- 🖼️ Product images (burger, cola, pizza)

## 🛠️ Technologies
- C#
- Windows Forms
- .NET Framework
